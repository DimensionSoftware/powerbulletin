mutant.js
=========

a system for node.js and express that unifies client and server page rendering in an SEO-friendly way

INSTALLATION
============

    npm install mutant

USAGE
=====

    var mutant = require('mutant');
    var mw     = require('mutant/middleware');

DEV
===

    npm install
    lsc --watch --compile *.ls &
